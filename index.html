HTML

  • The `src` attribute defines the location, or source, of the image file.
  • The `alt` attribute contains a text string that describes the appearance and functionality of the image.
  • The footer tag appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
  • The head element contains information about the webpage. It is also where the CSS file is linked.
  • The body element represents the visible content shown to the user. It is also where the JavaScript file is linked.
  • Switch ul to ol to make the list ordered.
  • Link element connects html to external file.
  • The period in the filepath means current directory.
  • rel attribute specifies the relationship between current document and linked document.
  • href is the location URL (filepath) of the external resource.

CSS

  • Block value assigns each image to start on a new line.
  • The period preceding the selector turns it into a class selector.
  • The box-shadow goes from X then Y ordered offsets. Ex: If the colon is first proceeded by 5px, 5px is the width (x-axis) of the shadow.

Git

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • git status: checks what branch we are currently on
  • git checkout -b [branch-name]: creates a new branch and switches to it

JavaScript

  • Note: The reason you place the script element near the bottom of the HTML file is that the browser reads code in the order it appears in the file. If the JavaScript loads first and it is supposed to affect the HTML that hasn't loaded yet, there could be problems. Placing JavaScript near the bottom of an HTML page is one way to accommodate this dependency.
  • The alert function, alert("text"); makes a pop-up dialogue box appear inside the browser window with the text written.
  • The prompt function, alert("Please enter your [ex: name]."); makes a pop-up dialogue box appear inside the browser window prompting the user to enter the information asked for.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
